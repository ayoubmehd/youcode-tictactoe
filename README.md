
# Tictactoe

This is multiplayer Tictactoe game build with react and socket io

## Run Locally

Clone the project

```copy
  git clone https://github.com/ayoubmehd/youcode-tictactoe
```

Go to the project directory

```copy
  cd youcode-tictactoe
```

Install dependencies

```copy
  yarn
```

Start the server

```copy
  yarn dev
```
